## 1. システム概要

### 1.1 目的
児童発達支援管理責任者（児発管）の業務支援を目的とした、生成AIによる個別支援計画書および支援プログラムの自動作成支援ツール。

### 1.2 主な機能

| 機能 | 内容 |
|------|------|
| 個別支援計画の自動作成 | アセスメントシートのPDFや入力情報から、5領域に基づく個別支援計画を作成 |
| 支援プログラムの自動作成 | 個別支援計画書のPDFをアップロードし、個別または集団の支援プログラムを作成 |
| 手書きPDF読み取り | 手書きのアセスメントシートも自動で読み取り可能 |
| PDF出力・コピー | 作成結果をPDF保存またはWord等にコピー貼り付け可能 |

### 1.3 動作環境

- 推奨ブラウザ: Google Chrome、Microsoft Edge（最新版）
- インターネット接続必須
- PCでの利用を推奨（スマホ・タブレットでも閲覧可能）

## 2. ログイン方法

### 手順

1. Webブラウザで本サービスのURL（管理者から共有）にアクセス
2. ログイン画面が表示されたら、配布されたパスワードを入力
3. 「ログイン」ボタンをクリック
4. メイン画面が表示されたらログイン成功

### ⚠️ 注意

- パスワードは業務関係者以外と共有しない
- 共用PCではログイン後、必ず画面右上の「ログアウト」をクリック
- セッションは8時間で自動的に切断

### パスワードを忘れた場合
管理者にお問い合わせください。

## 3. 個別支援計画の作成（タブ1）

画面上部の「**個別支援計画**」タブを選択します。

### 3.1 入力項目（上から順に）

#### ① 教室情報（任意）
- 教室名（例: ウィズ・ユー○○教室）
- 教室の特徴・方針

#### ② アセスメントシート【メイン入力・推奨】
- **PDFアップロード**: アセスメントシートをPDFで添付（手書きOK）
- **アセスメント補足**: PDFに含まれない情報（最近の様子、検査結果、他機関からの情報など）を追加で入力（任意）

> 💡 **推奨**: 児童に関する情報は **アセスメントシートのPDFに集約** してアップロードしてください。氏名・年齢・診断名・家族の意向・目標などはすべてアセスメントシートから自動で読み取ります。

#### ③ 過去の個別支援計画書（任意）
- 過去の計画書PDFをアップロードすると、内容との整合性を保った計画を作成
- 氏名・年齢・診断名等もPDFから自動で読み取られます

### 3.2 計画作成の実行

1. 必要な情報を入力（最低でも1項目は入力必要）
2. 画面下部の「**5領域に基づく支援計画を作成する**」ボタンをクリック
3. 1〜2分ほど待つ（PDFの読み取りと計画生成）
4. 結果が画面に表示される

### 3.3 出力される内容

| セクション | 内容 |
|-----------|------|
| 基本情報 | 氏名、年齢、診断名、受給者証番号など |
| 利用児及び家族の生活に対する意向 | 保護者からの聞き取りに基づく意向・希望 |
| 総合的な支援の方針 | 支援全体の方向性（箇条書き） |
| 長期目標・短期目標 | 1年・6ヶ月の達成目標 |
| 備考 | 加算関連の定型文 |
| 支援目標及び具体的な支援内容（5領域＋家族・移行・地域連携） | 各項目の達成目標・支援内容・担当者・優先度を表で整理。**各行の直下に「根拠」枠を併記**し、なぜその支援内容になったかを明示 |
| 特記事項 | 提供時間・延長支援時間は **伏字（〇〇時〇〇分）** で出力。施設で実際の時間を記入 |
| 延長を必要とする理由及び時間 | 定型文 |
| 意図・根拠まとめ | 長期目標／短期目標／5領域の重点配分など、計画全体の意図を簡潔に整理 |

### 3.4 出力結果の見方

支援目標の表では、以下の3つの工夫がされています：

1. **具体策の枠分け** — 1つのセル内で複数の具体策がある場合、それぞれが独立した枠（カード）で表示
2. **5領域バッジ** — 各具体策の右側に「健康や生活」「運動や感覚」など、対応する5領域名が小さなバッジで表示
3. **根拠行** — 各データ行のすぐ下に、薄い黄色背景の「根拠」枠があり、その目標・支援内容を選んだ理由が記載

> 💡 **編集のヒント**: 特記事項の時間部分（〇〇時〇〇分）は意図的に伏字になっています。Word/Excel に貼り付けた後で、各施設の実際の提供時間に合わせて記入してください。

## 4. 支援プログラムの作成（タブ2）

画面上部の「**支援プログラム**」タブを選択します。

### 4.1 機能の概要

作成済みの個別支援計画書のPDFをアップロードすることで、具体的な支援プログラムを自動作成。

| アップロード数 | 生成内容 |
|---------------|----------|
| **1名分** | 個別支援プログラム（週間プログラム表、1セッション構成例など） |
| **2名分以上** | 集団プログラム（共通ニーズ分析、集団で実施できる10案、個別配慮一覧など） |

### 4.2 入力項目

- **個別支援計画書（PDF）**: 必須。複数選択可（Ctrlキー押しながら選択）
- 教室名、教室の特徴・方針（任意）
- 1セッションの時間・頻度（例: 60分／週3回）
- 追加の指示・条件（運動系中心、季節行事を入れたい など）

### 4.3 プログラム作成の実行

1. 個別支援計画書のPDFを1〜複数選択してアップロード
2. 必要に応じて教室情報や追加指示を入力
3. 「**支援プログラムを作成する**」ボタンをクリック
4. 2〜3分ほど待つ
5. 結果が表示される

### 4.4 出力される内容

#### 個別支援プログラム（1名分）の場合
- 対象児童の情報
- プログラムのねらい
- 週間支援プログラム表（月〜日の7曜日、通所/家庭、5領域別）
- 1セッションの構成例（時間配分・声かけ例）
- 声かけ・関わり方のポイント
- 家庭・学校との連携
- 評価・見直しの視点

#### 集団プログラム（2名以上）の場合
- 対象児童一覧（5領域別の支援目標）
- 共通する支援ニーズの分析
- 集団プログラム概要
- 集団で実施できる支援プログラム10案（カード形式）
- 個別配慮一覧
- 評価の視点、準備物

## 5. 結果の保存・出力

作成結果は3つの方法で保存・利用できます。

### 5.1 PDF保存

1. 結果画面右上の「**PDF保存**」ボタンをクリック
2. 新しいウィンドウで印刷ダイアログが開く
3. 「送信先」または「プリンター」で「**PDFに保存**」を選択
4. 用紙サイズ・向きはA4横向きに自動設定
5. 「保存」をクリックしてファイル名・保存先を選択

### 5.2 コピー（Word/Excelへの貼り付け）

1. 結果画面右上の「**コピー**」ボタンをクリック
2. 「表ごとコピーしました」と表示されたら成功
3. Word・Excel・Googleドキュメント等で貼り付け（Ctrl+V）
4. 表もそのまま貼り付けられます

> 💡 メモ帳など書式に対応しないアプリでは、表はタブ区切り（TSV）で貼り付けられます。

### 5.3 リセット
「リセット」ボタンで入力フォームと結果をすべてクリアできます。

## 6. 個人情報の取り扱い・運用ルール

> ⚠️ **重要**
> 本システムは入力された情報を **Google Cloud社の Vertex AI（東京リージョン `asia-northeast1`）** に送信して処理します。
> データ処理は日本国内で行われますが、運営事業者（Google LLC）が米国法人のため、個人情報保護法第28条「外国第三者提供」に該当します。以下のルールを必ず守ってください。

### 6.1 必須ルール

1. **保護者の同意**: 本サービスでAI処理することについて、事前に保護者から同意を得る
2. **仮名運用の推奨**: 児童名・記入者名は可能な限り仮名（イニシャル等）で入力
3. **パスワードの管理**: 業務関係者以外と共有しない。書面・SNSでの送付禁止
4. **出力の確認**: 生成結果は素案。必ず児発管が内容を確認・修正してから業務利用

### 6.2 データの保管期間

| 保管場所 | 保管期間 |
|----------|----------|
| 本システムサーバー | 処理後即破棄（データベース保存なし） |
| セッション情報 | 最大8時間 |
| Vertex AI（東京リージョン・不正検知用） | 最大60日間 → 完全削除 |
| モデル学習 | 一切利用されない（Vertex AI規約上明示） |

### 6.3 推奨される入力方法

| 項目 | 推奨入力 | 避けるべき入力 |
|------|----------|---------------|
| 児童名 | `Aさん`、`T.K.`、`児童1` | フルネーム |
| 記入者名 | `母`、`保護者A` | フルネーム |
| 受給者証番号 | 未入力推奨 | 正確な番号 |
| 診断名 | そのまま入力可 | — |

## 7. トラブルシューティング

### 7.1 ログインできない

| 症状 | 対処 |
|------|------|
| パスワードが正しくありません | 大文字小文字、半角/全角に注意 |
| 15分後に再度お試しください | 10回連続で失敗するとロック。15分待つか管理者に連絡 |
| 画面が表示されない | キャッシュクリア（Ctrl+Shift+R）して再読み込み |

### 7.2 PDFアップロード時のエラー

| 症状 | 対処 |
|------|------|
| ファイルサイズは15MB以下 | PDFを圧縮するか、ページ数を減らす |
| PDFファイルのみ | JPG/PNG等の画像は事前にPDF化 |
| 読み取りエラー | パスワード保護されたPDFは解除してから再アップロード |

### 7.3 計画書の作成でエラー

| 症状 | 対処 |
|------|------|
| API Error: 429 | クォータ超過。1〜2分後に再試行 |
| サーバーエラー | 時間を空けて再試行 |
| 処理が長時間終わらない | 3分以上かかる場合はブラウザを再読み込み |
| リクエストが多すぎます | 1分間に30回以上のアクセス。少し待ってから |

### 7.4 出力結果の不具合

| 症状 | 対処 |
|------|------|
| 氏名が誤認識 | 手書きの場合誤読あり。補足欄で正しい情報を補足 |
| 内容が不適切・的外れ | 入力情報を増やして再生成、または手動修正 |
| 表が崩れて表示 | 結果画面は横スクロールで全体確認可能 |
| PDF保存で書式崩れ | 印刷ダイアログで「A4横向き」「拡大縮小: なし」 |
