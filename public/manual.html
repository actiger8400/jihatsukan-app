1. システム概要

1.1 本システムでできること

児童発達支援管理責任者（児発管）の業務を、生成AIがサポートします。

機能内容
個別支援計画の自動作成アセスメントシートのPDFや入力情報から、5領域に基づく計画書を作成
支援プログラムの自動作成個別支援計画書から、個別または集団のプログラムを作成
手書きPDF読み取り手書きのアセスメントシートも自動で読み取り可能
PDF・コピー出力結果をPDF保存またはWord/Excelに表ごとコピー

1.2 動作環境

  • 推奨ブラウザ: Google Chrome、Microsoft Edge（最新版）
  • インターネット接続必須
  • PCでの利用を推奨

推奨
児童に関する情報は アセスメントシートのPDF に集約してアップロードしてください。氏名・年齢・診断名・家族の意向・目標などはすべてアセスメントシートから自動で読み取ります。手書きのシートも自動読み取りに対応しています。
アセスメント補足欄では、PDFに含まれない情報（最近の様子、検査結果、他機関からの情報など）を追加でテキスト入力できます。

3.2 計画作成の実行

操作手順
  1. 必要な情報を入力（最低でも1項目は入力必要）
  2. 画面下部の「5領域に基づく支援計画を作成する」ボタンをクリック
  3. 1〜2分ほど待機（PDF読み取り＋計画生成）
  4. 結果が画面に表示される

3.3 出力される内容

セクション内容
基本情報氏名、年齢、診断名、受給者証番号など
利用児及び家族の生活に対する意向保護者からの意向・希望
総合的な支援の方針支援全体の方向性（箇条書き）
長期目標・短期目標1年・6ヶ月の達成目標
備考加算関連の定型文
支援目標及び具体的な支援内容（5領域＋家族・移行・地域連携）各項目の達成目標・支援内容・担当者・優先度を表で整理。各行の直下に「根拠」枠を併記し、なぜその支援内容になったかを明示
特記事項提供時間・延長支援時間は 伏字（〇〇時〇〇分） で出力。施設で手書き記入
延長を必要とする理由及び時間定型文
意図・根拠まとめ長期目標／短期目標／5領域の重点配分など、計画全体の意図を簡潔に整理

3.4 出力結果の見方

支援目標及び具体的な支援内容の表では、以下の3つの工夫がされています：
  1. 具体策の枠分け: 1つのセル内で複数の具体策がある場合、それぞれが独立した枠（カード）で表示されます
  2. 5領域バッジ: 各具体策の右側に「健康や生活」「運動や感覚」など、対応する5領域名が小さなバッジで表示されます
  3. 根拠行: 各データ行のすぐ下に、薄い黄色背景の「根拠」枠があり、その目標・支援内容を選んだ理由が記載されます
編集のヒント
特記事項の時間部分（〇〇時〇〇分）は意図的に伏字になっています。各施設の実際の提供時間に合わせて、Word/Excel に貼り付けた後で記入してください。

結果画面の右上に3つのボタンが表示されます。

5.1 PDF保存

PDFに保存する手順
  1. 「PDF保存」ボタンをクリック
  2. 新しいウィンドウで印刷ダイアログが開く
  3. 「送信先」または「プリンター」で「PDFに保存」を選択
  4. 用紙設定はA4横向きに自動設定済み
  5. 「保存」をクリックしてファイル名・保存先を選択

5.2 コピー（Word/Excel貼り付け）

コピーの手順
  1. 「コピー」ボタンをクリック
  2. 「表ごとコピーしました」と表示されたら成功
  3. Word・Excel・Googleドキュメント等で貼り付け（Ctrl+V）
  4. 表もそのまま貼り付けられる
メモ帳など書式に対応しないアプリでは、表はタブ区切り（TSV）形式で貼り付けられます。Excelでも各セルが分かれた状態で貼り付け可能です。

5.3 リセット

「リセット」ボタンで入力フォームと結果をすべてクリアできます。次の児童の計画作成を続けて行う際にご利用ください。

6. 個人情報の取り扱い

重要
本システムは入力情報を Google Cloud社の Vertex AI（東京リージョン） に送信して処理します。データ処理は日本国内で行われますが、運営事業者（Google LLC）が米国法人のため、個人情報保護法第28条「外国第三者提供」に該当します。以下のルールを必ず守ってください。

6.1 必須ルール

  1. 保護者の同意: AI処理について、保護者から事前同意を得る
  2. 仮名運用の推奨: 児童名・記入者名は可能な限り仮名（イニシャル等）
  3. パスワード管理: 業務関係者以外と共有しない
  4. 出力の確認: 必ず児発管が内容を確認・修正してから業務利用

6.2 データの保管期間

保管場所保管期間
本システムサーバー処理後即破棄（DB保存なし）
セッション情報最大8時間
Vertex AI（東京リージョン・不正検知用）最大60日間 → 完全削除
モデル学習一切利用されない（Vertex AI規約上明示）

7. トラブルシューティング

7.1 ログインできない

症状対処
パスワードが正しくありません大文字小文字、半角/全角を確認
15分後に再度お試しください10回失敗でロック。15分待つか管理者に連絡
画面が表示されないキャッシュクリア（Ctrl+Shift+R）

7.2 PDFアップロード時のエラー

症状対処
ファイルサイズは15MB以下PDFを圧縮 or ページ削減
PDFファイルのみJPG/PNG等は事前にPDF化
読み取りエラーパスワード保護を解除

7.3 計画書の作成でエラー

症状対処
API Error: 429（クォータ超過）1〜2分後に再試行
サーバーエラー時間を空けて再試行
処理が長時間終わらない3分以上ならブラウザを再読み込み
リクエストが多すぎます1分間に30回以上のアクセス。少し待つ

7.4 出力結果の不具合

症状対処
氏名が誤認識手書きの場合誤読あり。補足欄で正しい情報を補足
内容が不適切・的外れ入力情報を増やして再生成、または手動修正
表が崩れて表示結果画面は横スクロールで全体確認可能
PDF保存で書式崩れ印刷ダイアログで「A4横向き」「拡大縮小: なし」
